# AI Gmail Phishing Detector

## Overview
The AI Gmail Phishing Detector is a machine learning-based application that identifies phishing attempts in Gmail. This tool utilizes advanced algorithms to analyze incoming emails and warn users about potential threats.

## Features
- **Real-time Detection**: Analyzes emails as they arrive.
- **User-friendly Interface**: Easy to navigate and use.
- **Machine Learning Algorithms**: Utilizes state-of-the-art models for email classification.

## Getting Started

### Prerequisites
- Python 3.x
- Required libraries: `pandas`, `numpy`, `scikit-learn`, `flask`

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/muhammedmuflih/AI-Gmail-Phishing-Detector.git
   ```
2. Navigate to the project directory:
   ```
   cd AI-Gmail-Phishing-Detector
   ```
3. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```

### Usage
1. Run the Flask application:
   ```
   python app.py
   ```
2. Access the application in your web browser at `http://localhost:5000`.
